Como o computador funciona?

Primeira etapa!

Ao pressionar o botão de energia o computador disfere energia para seus componentes os possibilitando de iniciar o processo sabe aquela caixa grande que fica no quanto do pc? aquilo é conhecido como a fonte de energia é la onde o cabo energia é conectado.

Segunda etapa!

Com a eletricidade a CPU se inicia e procura por um programa que normalmente esta na placa mãe este programa é chamado de BIOS e UEFI, porém hoje em dia apenas o UEFI é ultilizado, eles carregam opções de configurações nas quais tstan e iniciam o hardware do seu sistema e isso enclui a sua propia CPU durante essa fase é comum ver a logo da empresas de fabricação do seu PC.

Terceira etapa!

O Bootloader então é iniciado, ele é um programa que tem o grande trabalho de ligar o resto do sistema de operação o bootloader do windows é chamado de Windows boot manager, a maioria do sistema do linux usa um chamado GRUB e o Mac usa um chamado de Boot

Quarta etapa!

Quando sistema estiver carregado e estiver na memória ele inicia isso inclui o carregamento de drivers do dispositivo,então finalmente quando a inicialização estiver pronta o sistema mostra uma prompt de login para o usuário .

Quinta etapa!

Interface gráfica de usuário ou GUI nesta parte é onde você vê uma área de trabalho com ícones, janelas de apps e a barra de tarefa, você pode usar o mouse para clicar em apps, arrastar e utilizar ferramentas que o próprio computador fornece.

Sexta etapa!

A iniciação de um app ao clicar em um app e o executá- lo o sistema operacional aloca recursos de hardware, como núcleos de CPU, memória ram e espaço de disco a CPU executa o código, a memória ram armazena dados e variáveis e o armazenamento de disco mantém os arquivos, então o programa é iniciado e exibido na tela para o usuário, permitindo a utilização do programa.
